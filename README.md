### Clone config
#### Windows
```Shell
git clone https://github.com/SummonHIM/my-config.nvim.git %LOCALAPPDATA%\nvim
```

#### Linux
```Shell
git clone https://github.com/SummonHIM/my-config.nvim.git ~/.config/nvim
```
